## 📖 Overview

The **Multimodal Authentication System** enhances security by combining two biometric authentication methods:

👤 **Face Recognition**  
🎙️ **Voice Authentication**

Instead of relying on a single authentication mechanism, the system validates users through multiple biometric inputs, improving accuracy and minimizing unauthorized access.

This project leverages **Machine Learning**, **Computer Vision**, and **Speech Processing** techniques for secure user verification.

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/Aashish-Anderson/Multimodal-Authentication-System.git
```

Move into project:

```bash
cd Multimodal-Authentication-System
```

Create virtual environment:

```bash
python -m venv .venv
```

Activate environment:

**Windows**

```bash
.venv\Scripts\activate
```

**Linux/Mac**

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Project

Run application:

```bash
python app.py
```

or

```bash
python main.py
```
